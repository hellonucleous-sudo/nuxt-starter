5. Webhook & API Reference

    

      

        

          Use these API endpoints to push leads from any external tool — Zapier, Make, n8n, or custom code.
          Replace YOUR_URL with your Apps Script deployment URL.
        

        

          

            
// Add a lead (GET)

            GET YOUR_URL?fn=addLead&args=[{"name":"John","phone":"9999999999","email":"john@co.com","source":"Website"}]
          

          

            
// Add a lead (POST)

            POST YOUR_URL

            Content-Type: application/json

            {"fn":"addLead","args":[{"name":"Jane","phone":"8888888888","source":"Zapier"}]}
          

          

            
// Get all leads

            GET YOUR_URL?fn=getLeads
          

        

        

          All endpoints return JSON. CORS is open. No authentication token needed — security is through obscurity of the URL.
          For production, consider deploying with restricted access if needed.
